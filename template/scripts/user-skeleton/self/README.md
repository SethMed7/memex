# self/ — the whole-subject layer

Slow-changing facts about *who or what* this memex is for — a person, an assistant, a team, a system.
Numbered notes keep a natural read order; add as many as you need. Each is prose-first with a one-line
summary near the top so `MAP.md` can index it. This is the layer an assistant should treat as durable
ground truth.

Starter: `00-identity.md`. Add more as fits your subject (context, principles, goals, history, …).
